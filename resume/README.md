# Resume Folder

Place your resume PDF file in this folder with the filename: `Yaksh_Verma_Resume.pdf`

## Instructions

1. Export your resume as a PDF file
2. Rename it to `Yaksh_Verma_Resume.pdf`
3. Place it in this `resume/` folder
4. The resume will automatically appear on your portfolio website

## File Requirements

- **Format**: PDF (`.pdf`)
- **Filename**: `Yaksh_Verma_Resume.pdf` (exact match, case-sensitive)
- **Location**: This folder (`resume/Yaksh_Verma_Resume.pdf`)

## Features

Once uploaded, visitors to your portfolio will be able to:
- View the resume in an embedded PDF viewer
- Download the resume directly
- Open the resume in a new tab for full-screen viewing
